# While Loop Tasks in C++
This repository contains a collection of 18 basic programming tasks solved using **While Loops** in C++.

## Tasks Included:
1. Print numbers 1 to N
2. Reverse counting N to 1
3. Even and Odd numbers
4. Multiples of 2, 5, and K
...
5. Skipping multiples of 4

## How to use:
Each task is documented in the main `.cpp` file. To run a specific task, simply uncomment the `main()` function for that task.
